# 🎧 کتابخانه صوتی — Telegram Mini App

یک مینی‌اپ تلگرام برای پادکست کتاب، با رابط کاربری حرفه‌ای شبیه اسپاتیفای.

---

## 📁 ساختار فایل‌ها

```
podcast-miniapp/
├── index.html          ← صفحه اصلی اپ
├── css/
│   └── style.css       ← تمام استایل‌ها
├── js/
│   └── app.js          ← منطق اصلی برنامه
├── data/
│   └── catalog.js      ← 📌 اینجا کتاب‌ها و اپیزودها را اضافه کنید
└── README.md
```

---

## 🚀 راه‌اندازی روی GitHub Pages

### مررحله ۱ — ساخت ریپو
1. وارد [github.com](https://github.com) شوید
2. روی **New repository** کلیک کنید
3. نام مثلاً `podcast-app` بگذارید
4. **Public** را انتخاب کنید
5. روی **Create repository** کلیک کنید

### مرحله ۲ — آپلود فایل‌ها
1. روی **Add file > Upload files** کلیک کنید
2. تمام فایل‌ها و پوشه‌ها را آپلود کنید
3. روی **Commit changes** کلیک کنید

### مرحله ۳ — فعال کردن GitHub Pages
1. به **Settings > Pages** بروید
2. زیر **Branch**، گزینه `main` را انتخاب کنید
3. روی **Save** کلیک کنید
4. چند دقیقه صبر کنید

🔗 آدرس اپ شما می‌شود:
```
https://USERNAME.github.io/REPO-NAME/
```

### مرحله ۴ — ساخت بات تلگرام
1. به [@BotFather](https://t.me/BotFather) پیام بدید
2. `/newbot` بفرستید و نام بات را بدید
3. `/newapp` بفرستید (یا `/setmenubutton`)
4. آدرس GitHub Pages را به عنوان Web App URL وارد کنید

## ➕ چطور کتاب یا اپیزود جدید اضافه کنیم؟

### فایل `data/catalog.js` را ویرایش کنید

#### اضافه کردن کتاب جدید:
```javascript
{
  id: "book-4",                    // ← شناسه یکتا (book-N)
  title: "نام کتاب",
  author: "نام نویسنده",
  category: "دسته‌بندی",           // ← در فیلتر نمایش داده می‌شود
  description: "توضیحات کوتاه...",
  totalEpisodes: 1,
  episodes: [
    {
      id: "ep-4-1",                // ← شناسه یکتا (ep-N-M)
      title: "عنوان اپیزود",
      duration: "35:00",           // ← مدت زمان (نمایشی)
      audioUrl: "https://t.me/YOUR_CHANNEL/POST_ID",  // ← لینک پست تلگرام
      description: "توضیح کوتاه اپیزود",
      date: "۱۴۰۳/۰۴/۰۱",
      isNew: true                  // ← true = نمایش برچسب «جدید»
    }
  ]
}
```

#### اضافه کردن اپیزود به کتاب موجود:
فقط یک آبجکت به آرایه `episodes` کتاب مورد نظر اضافه کنید:
```javascript
{
  id: "ep-1-4",
  title: "فصل چهارم: ...",
  duration: "40:00",
  audioUrl: "https://t.me/your_channel/8",
  description: "توضیحات...",
  date: "۱۴۰۳/۰۴/۱۵",
  isNew: true
}
```

### ✅ پس از ویرایش:
1. فایل `catalog.js` را در GitHub آپلود/ویرایش کنید
2. ۱-۲ دقیقه صبر کنید
3. اپ به‌صورت خودکار به‌روز می‌شود!

## 🔗 چطور لینک صوتی از تلگرام بگیریم؟

1. پست صوتی را در کانال خود آپلود کنید
2. روی پست **راست‌کلیک** (یا نگه داشتن) کنید
3. **Copy link** را بزنید
4. لینک را در `audioUrl` قرار دهید

📌 نکته: لینک به شکل `https://t.me/channel_name/12` است.

اگر کانال خصوصی دارید، می‌توانید فایل را **forward** کنید به یک کانال عمومی جداگانه و لینک آن را استفاده کنید.

## ✨ قابلیت‌ها

- 📚 نمایش کتاب‌ها به صورت کارت‌های گرافیکی
- 🔍 جستجوی آنی در کتاب‌ها و نویسنده‌ها
- 🏷 فیلتر بر اساس دسته‌بندی
- 🎙 لیست اپیزودهای هر کتاب
- 🎧 پلیر شبیه اسپاتیفای (کوچک + گسترده)
- ⏱ کنترل ۱۵ ثانیه جلو/عقب
- ⚡ کنترل سرعت پخش (۰.۷۵× تا ۲×)
- 🔗 باز کردن فایل اصلی در تلگرام
- 🌙 تم تاریک زیبا
- 📱 کاملاً ریسپانسیو برای موبایل
